IPinfo 提供的IP查询接口

请求URL：https://him.cm/tool/api.php

请求方式：GET

请求参数：

参数名 是否必须 类型 说明
ip 否 string 默认返回访客自身IP，否则返回指定IP
data 否 string ip:仅返回IP;addr:仅返回地址; 默认返回ip + 地址
type 否 string 默认返回文本格式; json：返回json格式

返回示例：

api.php?ip=180.97.33.108&type=json 返回

{
    "status": 1,
    "ip": "180.97.33.108",
    "addr": "江苏省南京市 北京百度网讯科技有限公司电信节点"
}


api.php?ip=180.97.33.108&type=text 返回

180.97.33.108 江苏省南京市 北京百度网讯科技有限公司电信节点


错误的示例：

https://him.cm/tool/api.php?ip=errip 返回

{"status":0,"msg":"IP\u683c\u5f0f\u4e0d\u5bf9"}
